setHolidays

fun setHolidays(ccy: String, holidays: Set<LocalDate>): Unit

Registers the holidays of the given currency.

This method does not defensively normalize the given currency to lower- or upper-case when registering holidays. It also does not defensively copy the given set of holidays when registering.

Parameters

ccy - the currency for registering the holidays with

holidays - the set of dates designated as holidays for the given currency
