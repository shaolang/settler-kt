isHoliday

fun isHoliday(ccy: String, date: LocalDate): Boolean

Determines whether the given date is a holiday for the given currency.

This method does not defensively normalize the given currency to lower- or upper-case when checking.

Parameters

ccy - the currency to determine whether the date is a holiday

date - the date to determine whether it's a holiday for the given currency

Return
true when the currency and date combination is a registered holiday in the registry
